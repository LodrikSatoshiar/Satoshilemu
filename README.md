# Satoshilemu

Una economía circular basada en Bitcoin, nacida en Pichilemu, Chile.

## 🌊 ¿Qué es Satoshilemu?

Satoshilemu es un experimento de soberanía económica local que utiliza Bitcoin como base para construir una economía circular real.

No es solo software.  
Es un protocolo social.

## ⚡ Principios

- Soberanía individual
- Dinero sin intermediarios
- Comunidad local fuerte
- Circularidad económica
- Adopción desde la primera línea

## 🔁 ¿Cómo funciona?

- Distribución periódica de renta básica en Bitcoin
- Comercios locales que aceptan pagos en Bitcoin
- Infraestructura abierta basada en Lightning Network
- Incentivos para circular el dinero dentro de la comunidad

## 🧩 Componentes

- Infraestructura de pagos (BTCPay Server, Lightning)
- Sistema de distribución de fondos
- Gobernanza comunitaria
- Playbooks replicables para otras ciudades

## 🌍 Replicabilidad

Satoshilemu está diseñado como un modelo abierto que puede ser implementado en otras ciudades.
