# Contribuir a Satoshilemu

Gracias por tu interés en contribuir.

## Principios

- Mantener el enfoque en Bitcoin
- Priorizar soluciones simples y robustas
- Pensar en replicabilidad

## Cómo contribuir

1. Fork del repositorio
2. Crear branch (`feature/...`)
3. Pull request con explicación clara

## Importante

Este proyecto usa licencia GPL v3.  
Toda contribución será liberada bajo la misma licencia.
